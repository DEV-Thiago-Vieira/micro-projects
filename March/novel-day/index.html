Novel Day

The goal is to practice manual skills without any assistance of AI. The writing process should be customizable and fast. The user should be able to use GPT automatically to check the grammar. You should be able to quickly adjust tests, the back end should store the data and save accounts.

This project should be cheap and used mostly for personal use. However it can escalate to a larger project to automatize chapters to other platforms. But in essence is a project to assist the user in his goals, give feedback about the writing, automatic proofreading, and all the conveniences a writer could need.

At the same time I plan to develop a method using this site to plan using a simple framework.

What this app does

    Features:
  • Track your daily writing progress
  • Plan your writing schedule
  • Write and organize your novel
